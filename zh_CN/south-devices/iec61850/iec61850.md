# IEC61850 介绍与使用

## 模块描述

iec61850 插件用于对IEC61850服务器的读/写，目前只支持MMS协议的访问。

## 参数配置

| 字段     | 说明                       |
| -------- | -------------------------- |
| **host** | 远程设备的 IP              |
| **port** | 远程设备的端口，默认为 102 |

## 支持的数据类型

* INT8
* UINT8
* INT16
* UINT16
* INT32
* UINT32
* INT64
* UINT64
* FLOAT
* DOUBLE
* BOOL
* STRING

## 地址格式用法

### 地址格式

> Logical Devices/Logical Nodes$FC$DO$DA</span>
