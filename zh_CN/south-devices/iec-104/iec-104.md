# IEC 60870-5-104 介绍与使用

## 参数配置

| 字段          | 说明                  |
| ------------ | --------------------- |
| **host**     | 设备 IP                |
| **port**     | 设备端口号，默认为 2404  |
| **ca**       |  公共地址               |
| **interval** | 站点问询时间间隔         |

### 支持的数据类型

* UINT16
* INT16
* FLOAT
* BIT
* UINT8
* INT8

### 地址格式

> IOA</span>

| IEC 60870-5-104  类型 ID         | 数据类型  |
| :------------------------------ | :------------ |
| M_ME_NB_1、M_ME_TE_1            | uint16/int16 |
| M_ME_NC_1、M_ME_TF_1            | float        |
| M_SP_NA_1、M_SP_TB_1            | bit          |
| M_ME_NA_1、M_ME_TD_1、M_ME_ND_1 | uint16/int16 |
