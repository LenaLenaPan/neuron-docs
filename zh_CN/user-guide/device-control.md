# 反控设备

## 第一步，界面更改值

当 Tag 设置了写属性时，数据监控界面的 Tag 会有一个写操作，点击 `Write` 可以实现反控设备，例如，修改具有写属性的 1!40001 点位地址的值，如下图所示。

![write](./assets/write.png)

* 单击要改值的标签末尾的`写`按键；
* 选择是否以十六进制方式输入，不选择；
* 输入标签新值，例如，123；
* 单击`提交`按键提交新的值。

::: tip
设备中该点位也必须具有可写属性，否则无法写成功。
:::
